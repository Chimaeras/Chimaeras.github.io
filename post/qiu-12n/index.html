求1+2+...+n

2021-09-07

求 1+2+...+n ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

  • 方法1：数学公式

    sum=((n+1)∗n)/2sum = ((n+1)*n)/2 sum=((n+1)∗n)/2

    class Solution {
    public int sumNums ( int n ) {
        int res = ( int ) Math.pow( n, 2 ) + n;
        return res >> 1;
    }
}

  • 方法2：递归调用

f(n)=n+n−1+n−2+...+1=>f(n)=f(n−1)+nf(n)=n+n-1+n-2+...+1=> f(n)=f(n-1)+n f(n)=n+n−1+n−2+...+1=>f(n)=f(n−1)+n

class Solution {
    public int sumNums ( int n ) {
        if ( n == 1 ) {
            return n;
        }
        return sumNums( n - 1 ) + n;
    }
}
