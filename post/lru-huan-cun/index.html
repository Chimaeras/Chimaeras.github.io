LRU缓存

2021-09-14

设计和构建一个“最近最少使用”缓存，该缓存会删除最近最少使用的项目。缓存应该从键映射到值(允许你插入和检索特定键对应的值)，并在初始化时指定最大容量。当缓存被填满时，它应该删除最近最少使用的项目。

它应该支持以下操作： 获取数据 get 和 写入数据 put 。

获取数据 get(key) - 如果密钥 (key) 存在于缓存中，则获取密钥的值（总是正数），否则返回 -1。
写入数据 put(key, value) - 如果密钥不存在，则写入其数据值。当缓存容量达到上限时，它应该在写入新数据之前删除最近最少使用的数据值，从而为新的数据值留出空间。

LRUCache cache = new LRUCache( 2 /* 缓存容量 */ );

cache.put(1, 1);
cache.put(2, 2);
cache.get(1);       // 返回  1
cache.put(3, 3);    // 该操作会使得密钥 2 作废
cache.get(2);       // 返回 -1 (未找到)
cache.put(4, 4);    // 该操作会使得密钥 1 作废
cache.get(1);       // 返回 -1 (未找到)
cache.get(3);       // 返回  3
cache.get(4);       // 返回  4

  • 方法1：map+双向队列

定义map和双向队列节点，使用伪开头和伪节点来指向开头和结尾，实际操作过程就比较好操作。同时准备几个方法：

  • moveToHead ：将节点移动到开头
  • removeTail：删除尾节点
  • removeNode：删除某个节点
  • addToHead：将某个节点添加到开头

判断流程：

get方法：

通过map获取到节点，

  • 如果节点不存在，则返回-1；
  • 存在则调用moveToHead方法，将当前node移至开头

put方法：

通过map获取到节点，

  • 如果key不存在，则创建一个新的节点，将其添加进map并移动到链表头部。
    • 判断链表长度，如果超出容量，则删除链表尾节点
    • 同时要删除map中的对应项
    • size--
  • 如果key存在，则修改节点value，同时将其移到头部。
class LRUCache {
    class dLinkedNode {
        int key;
        int value;
        dLinkedNode prev;
        dLinkedNode next;

        public dLinkedNode ( ) {
        }

        public dLinkedNode ( int _key, int _value ) {
            key = _key;
            value = _value;
        }
    }

    private Map< Integer, dLinkedNode > cache = new HashMap< Integer, dLinkedNode >();
    private int size;
    private int capacity;
    private dLinkedNode head, tail;

    public LRUCache ( int capacity ) {
        this.size = 0;
        this.capacity = capacity;
        // 使用伪头部和伪尾部节点
        head = new dLinkedNode();
        tail = new dLinkedNode();
        head.next = tail;
        tail.prev = head;
    }

    public int get ( int key ) {
        dLinkedNode node = cache.get( key );
        if ( node == null ) {
            return -1;
        }
        // 如果 key 存在，先通过哈希表定位，再移到头部
        moveToHead( node );
        return node.value;
    }

    public void put ( int key, int value ) {
        dLinkedNode node = cache.get( key );
        if ( node == null ) {
            // 如果 key 不存在，创建一个新的节点
            dLinkedNode newNode = new dLinkedNode( key, value );
            // 添加进哈希表
            cache.put( key, newNode );
            // 添加至双向链表的头部
            addToHead( newNode );
            ++size;
            if ( size > capacity ) {
                // 如果超出容量，删除双向链表的尾部节点
                dLinkedNode tail = removeTail();
                // 删除哈希表中对应的项
                cache.remove( tail.key );
                --size;
            }
        } else {
            // 如果 key 存在，先通过哈希表定位，再修改 value，并移到头部
            node.value = value;
            moveToHead( node );
        }
    }

    private void addToHead ( dLinkedNode node ) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }

    private void removeNode ( dLinkedNode node ) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    private void moveToHead ( dLinkedNode node ) {
        removeNode( node );
        addToHead( node );
    }

    private dLinkedNode removeTail ( ) {
        dLinkedNode res = tail.prev;
        removeNode( res );
        return res;
    }
}

  • 方法2：两个map分别维护value和time

我们使用分别使用两个Hashmap来存储不同的健值对，分别命名为：map、count。

map记录key-value

count记录key-time，time表示距离上一次访问的时间（从0开始）

get操作

  • 先判断是否已经存在于map中，如果是的话，用res将结果存储下来。
    • 然后我们要更新count中time：将所有key的time+1，将最近访问的key-time重置为0。
  • 如果不存在，返回-1

put操作

  • 看map中是否已经存在key，是的话直接覆盖。
  • 如果不存在，再看map是否已经满了，是的话，找出count中time最大的key，在map中将它移除，再加入最新的key-value。
  • 若都不满足，则表示key在map中不存在，且map没满，所以可以直接插入。

PS：在上述的每一步分支中，我们都要在结尾完成更新count中time的操作

class LRUCache {

    HashMap< Integer, Integer > map;
    HashMap< Integer, Integer > count;
    int capacity;
    int size = 0;

    public LRUCache ( int capacity ) {
        // map存储key-value
        map = new HashMap<>( capacity );
        // map容量
        this.capacity = capacity;
        // count存储key-time
        count = new HashMap<>( capacity );
    }

    public int get ( int key ) {
        int res = -1;
        // 如果map中已经存在key
        if ( map.containsKey( key ) ) {
            // res存储key对应的value
            res = map.get( key );
            // 将所有key的time+1
            count.replaceAll( ( n, v ) -> v + 1 );
            // 将count中的key次数重置为0，表示最近刚刚访问过
            count.put( key, 0 );
        }
        return res;
    }

    public void put ( int key, int value ) {
        // 看map中是否已经存在key，存在的话直接替换
        if ( map.containsKey( key ) ) {
            map.put( key, value );
        }
        // map中不存在，再看map是否满，满则需要进行替换
        else if ( size >= capacity ) {
            // 遍历一遍 取time最大的key
            int max = -1, maxKey = -1;
            for ( Map.Entry< Integer, Integer > entry : count.entrySet() ) {
                if ( max < entry.getValue() ) {
                    max = entry.getValue();
                    maxKey = entry.getKey();
                }
            }
            // 将time最大的key从map中移除
            map.remove( maxKey );
            count.remove( maxKey );
            // 将新添加的key-value放进去
            map.put( key, value );
        }
        // 即key中原先不存在，且map没满
        else {
            map.put( key, value );
            size++;
        }
        count.replaceAll( ( n, v ) -> v + 1 );
        count.put( key, 0 );
    }
}
