滑动窗口的最大值

2021-09-07

给定一个数组 nums 和滑动窗口的大小 k，请找出所有滑动窗口里的最大值。（你可以假设K总是有效的，即1<=k<=size）

输入: nums = [1,3,-1,-3,5,3,6,7], 和 k = 3
输出: [3,3,5,5,6,7] 
解释: 

  滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7

  • 方法1：暴力循环

用一个外循环来遍历整个数组，用一个内循环来模拟滑动窗口，在内循环中找出滑动窗口中的最大值，将其添加到数组中。

class Solution {
    public int[] maxSlidingWindow ( int[] nums, int k ) {

        if ( nums.length == 0 ) {
            return nums;
        }

        int size = nums.length;
        int[] res = new int[size - k + 1];
        int index = 0;

        for ( int i = 0; i <= size - k; i++ ) {
            int m = nums[i];
            for ( int j = i; j < i + k; j++ ) {
                m = Math.max( m, nums[j] );
            }
            res[index++] = m;
        }
        return res;
    }
}

  • 方法2：动态队列

可以使用动态队列来处理，有几点要注意的：

  • 队列内只包含窗口内的元素，当窗口进行移动的时候，需要将上一个窗口的左临界值（nums [ i - 1 ] ）去除。
  • 队列内保证顺序是从大到小，每轮窗口进行滑动的时候，都会添加一个新的值（nums [ j + 1 ]），这时候我们要将所有小于当前新值的元素进行删除，确保队列内顺序。
class Solution {
    public int[] maxSlidingWindow ( int[] nums, int k ) {
        if ( nums.length == 0 || k == 0 ) {
            return new int[0];
        }

        Deque< Integer > deque = new LinkedList<>();
        int[] res = new int[nums.length - k + 1];

        // 形成窗口前
        for ( int i = 0; i < k; i++ ) {
            while ( !deque.isEmpty() && deque.getLast() < nums[i] ) {
                deque.removeLast();
            }
            deque.addLast( nums[i] );
        }

        res[0] = deque.getFirst();

        // 形成窗口后
        for ( int i = k; i < nums.length; i++ ) {
            if ( deque.getFirst() == nums[i - k] ) {
                deque.removeFirst();
            }
            while ( !deque.isEmpty() && deque.getLast() < nums[i] ) {
                deque.removeLast();
            }
            deque.addLast( nums[i] );
            res[i - k + 1] = deque.getFirst();
        }

        return res;

    }
}
