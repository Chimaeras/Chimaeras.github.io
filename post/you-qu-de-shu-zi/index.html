有趣的数字

2021-09-30

题目描述

有n个数，两两组成二元组，相差最小的有多少对？相差最大的有多少对？

输入包含多组测试数据。
对于每组测试数据：
 	N - 本组测试数据有n个数
 	a1,a2...an - 需要计算的数据
保证:
 	1<=N<=100000,0<=ai<=INT_MAX. 
对于每组数据，输出两个数，第一个数表示差最小的对数，第二个数表示差最大的对数。

6
45 12 45 32 5 6
     
1 2

方法1：

首先基础判断可知复杂度为O(n^2)，可能会超时。所以先进行排序O（nlogn）。

  • 排序

  • 如果所有数字相等，直接利用公式计算：

Cmax=Cmin=(n∗(n−1))/2C_{max}=C_{min}=(n*(n-1))/2 Cmax​=Cmin​=(n∗(n−1))/2

  • 计算最小差数量：
    • 如果有重复数字，最小差必定是0
      • 在map中找到重复的数字出现的次数，利用公式计算
    • 如果没有重复的数字
      • 遍历查找最小差
  • 计算最大差数量：
    • 最大差的数量等于最大值的数量*最小值的数量。
package com.joker;

import java.io.Serializable;
import java.util.*;

/**
 * @author Joker
 */
public class Main {
    public static void main ( String[] args ) {
        Scanner input = new Scanner( System.in );
        while ( input.hasNext() ) {
            // 数组长度
            int n = input.nextInt();
            // 数据
            int[] data = new int[n];
            for ( int i = 0; i < n; i++ ) {
                data[i] = input.nextInt();
            }
            // 排序 否则（n^2）超时
            Arrays.sort( data );
            //如果数组中的值全相同，直接两两组合
            if ( data[0] == data[n - 1] ) {
                int tmp = ( n * ( n - 1 ) ) / 2;
                System.out.println( tmp + " " + tmp );
                continue;
            }
            // map存储数字出现的次数
            // hashmap和treemap结果不一样，为啥？
            Map< Integer, Integer > map = new TreeMap< Integer, Integer >();
            for ( int d : data ) {
                map.put( d, map.getOrDefault( d, 0 ) + 1 );
            }
            // 计算差最小的对数
            int minCount = 0;
            // 如果没有重复数字
            if ( map.size() == n ) {
                // 遍历计算差值
                int min = Math.abs( data[1] - data[0] );
                for ( int i = 2; i < n; i++ ) {
                    int temp = Math.abs( data[i] - data[i - 1] );
                    // 如果存在相同差值，则对数增加
                    if ( min == temp ) {
                        minCount++;
                    } else if ( temp < min ) {
                        // 出现更小的差，更新
                        min = temp;
                        minCount = 1;
                    }
                }
            } else {
                // 有重复数字，最小值一定是0，直接计算重复数字的个数
                for ( Integer key : map.keySet() ) {
                    int value = map.get( key );
                    if ( value > 1 ) {
                        minCount += ( value * ( value - 1 ) ) / 2;
                    }
                }
            }

            // 计算最大差
            int maxCount = 0;
            // 用list存储数组的值（从大到小）
            List< Integer > list = new ArrayList< Integer >( map.keySet() );
            int val1 = map.get( list.get( 0 ) );
            int val2 = map.get( list.get( list.size() - 1 ) );
            // 最大差的对数就等于最大值的数量*最小值的数量
            maxCount = val1 * val2;

            System.out.println( minCount + " " + maxCount );
        }

    }
}
