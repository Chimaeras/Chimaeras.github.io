用两个栈模拟队列

2021-09-11

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

  • 方法1：辅助栈

A作为主要栈，当元素进队时，压入A栈中；当删除队尾元素时，将A栈元素转移到B栈中，删除B栈栈顶元素，再jiangB栈元素转移回A栈。

class CQueue {

    Stack< Integer > A;
    Stack< Integer > B;

    public CQueue ( ) {
        A = new Stack< Integer >();
        B = new Stack< Integer >();
    }

    public void appendTail ( int value ) {
        A.push( value );
    }

    public int deleteHead ( ) {
        while ( !A.isEmpty() ) {
            B.push( A.pop() );
        }
        int res = B.isEmpty() ? -1 : B.pop();
        while ( !B.isEmpty() ) {
            A.push( B.pop() );
        }
        return res;
    }
}

  • 方法2：维护删除队列

  • 添加元素时，直接压入栈A

  • 删除元素时，若栈B为空，将栈A所有元素呈倒序压入栈B，这样栈B从顶部往下数，就是一个待删除的队列；若栈B部位口感，则删除顶部元素。这时就算期间穿插这添加元素的操作，也是把元素压入栈A中，对栈B没有影响。当栈B当前所有元素都出栈了，又将栈A的元素 ” 倒 “ 进来

class CQueue {

    Stack< Integer > A;
    Stack< Integer > B;

    public CQueue ( ) {
        A = new Stack< Integer >();
        B = new Stack< Integer >();
    }

    public void appendTail ( int value ) {
        A.push( value );
    }

    public int deleteHead ( ) {
        if ( !B.isEmpty() ) {
            return B.pop();
        } else {
            while ( !A.isEmpty() ) {
                B.push( A.pop() );
            }
        }
        return B.isEmpty() ? -1 : B.pop();
    }
}
