各位相加

2021-09-18

题目：给定一个非负整数num，反复将各个位上的数字相加，直到结果为一位数

  • 解法1：

class Solution {
    public int addDigits ( int num ) {
        int a, b, t;
        if ( num < 10 ) {
            return num;
        } else {
            a = num / 10;
            b = num % 10;
            t = addDigits( a ) + b;
            return addDigits( t );
        }
    }
}
  • 分析：a记录num除个位以外的数字，b记录num个位的数字，t记录各个位相加后的数字

  • 解法2：

class Solution {
    public int addDigits ( int num ) {
        return ( num - 1 ) % 9 + 1;
    }
}
