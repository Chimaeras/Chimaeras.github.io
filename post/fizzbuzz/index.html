FizzBuzz

2021-08-27

写一个程序，输出从1到n数字的字符串表示

  1. 如果 n 是3的倍数，输出“Fizz”；
  2. 如果 n 是5的倍数，输出“Buzz”；
  3. 如果 n 同时是3和5的倍数，输出 “FizzBuzz”。

  • 方法1：遍历判断

1、初始化一个列表存储结果字符串。

2、遍历1到n。

3、对于当前数，判断是否能被3和5整除，可以的话将"FizzBuzz"添加进答案。

4、判断能否被3整除，可以的话将“Fizz”添加进答案。

5、判断能否被5整除，可以的话将“Buzz”添加进答案。

6、以上都不行的话，将当前数字添加进答案。

class Solution {
    public List< String > fizzBuzz ( int n ) {
        //定义字符串List，作为结果
        List< String > res = new ArrayList< String >();

        //遍历
        for ( int i = 1; i <= n; i++ ) {
            boolean div3 = ( i % 3 == 0 );
            boolean div5 = ( i % 5 == 0 );

            if ( div3 && div5 ) {
                res.add( "FizzBuzz" );
            } else if ( div3 ) {
                res.add( "Fizz" );
            } else if ( div5 ) {
                res.add( "Buzz" );
            } else {
                res.add( Integer.toString( i ) );
            }
        }
        return res;
    }
}

  • 方法2：连接法

当规则变得更加复杂的时候，方法1需要不断增加判断条件分支，故方法2采用连接法。

1、 判断是否能被3整除，可以的话将Fizz连接进答案。

2、判断是否能被5整除，可以的话将Buzz连接进答案。

如此一来，当规则不断增加，也只需要添加相应的判断条件。

class Solution {
    public List< String > fizzBuzz ( int n ) {
        //定义字符串List，作为结果
        List< String > res = new ArrayList< String >();

        //遍历
        for ( int i = 1; i <= n; i++ ) {
            boolean div3 = ( i % 3 == 0 );
            boolean div5 = ( i % 5 == 0 );

            String ans = "";

            if ( div3 ) {
                ans += "Fizz";
            }
            if ( div5 ) {
                ans += "Buzz";
            }
            if ( "".equals( ans ) ) {
                ans += Integer.toString( i );
            }
            res.add( ans );
        }
        return res;
    }
}

  • 方法3：散列表法

当我们需要变换映射的时候，再用条件去判断就显得臃肿且难以维护。所以我们可以将对应关系放入散列表中，使其一一对应，当需要修改规则的时候，只需要修改散列表即可。

1、将映射关系放入散列表中。

2、遍历1到n。

3、对于每个数，遍历散列表中的键，检查是否能被整除。

4、如果能被整除，则将映射的值添加到答案中。

5、将答案添加到列表中。

class Solution {
    public List< String > fizzBuzz ( int n ) {
        List< String > res = new ArrayList< String >();

        HashMap< Integer, String > map = new HashMap< Integer, String >() {
            {
                put( 3, "Fizz" );
                put( 5, "Buzz" );
            }
        };
        for ( int i = 1; i <= n; i++ ) {
            String ans = "";

            // 遍历每个键，查看是否能被整除
            for ( Integer key : map.keySet() ) {
                if ( i % key == 0 ) {
                    ans += map.get( key );
                }
            }

            // 都不能被整除
            if ( "".equals( ans ) ) {
                ans += Integer.toString( i );
            }
            res.add( ans );
        }
        return res;
    }
}
