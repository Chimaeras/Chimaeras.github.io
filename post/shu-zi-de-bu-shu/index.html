数字的补数

2021-09-06

给你一个 正 整数 num ，输出它的补数。补数是对该数的二进制表示取反。

  • 解法1：

每当t向左移一位，c向右移一位，得到一串11111...的数。再令c与t进行抑或运算，一样为1的取0，不同的取1，相当于取反。

class Solution {
    public int findComplement ( int num ) {
        int t = num, c = 0;
        while ( t > 0 ) {
            t = t >> 1;
            c = ( c << 1 ) + 1;
        }
        return num ^ c;
    }
}

  • 解法2：

用a[i]存储二进制数组，再将二进制数组还原成十进制

class Solution {
    public int findComplement ( int num ) {
        int[] a = new int[32];
        int res = 0;
        for ( int i = 0; num > 0; i++ ) {
            a[i] = ( num % 2 == 1 ) ? 0 : 1;
            num /= 2;
        }
        for ( int i = 31; i >= 0; i-- ) {
            res += a[i] * Math.pow( 2, i );
        }
        return res;
    }
}
