两个链表的公共节点

2021-09-12

输入两个链表，找出它们的第一个公共节点。

  • 方法1：

两个链表的长度分别为a、b，使两个头节点都遍历（a+b）次，若存在公共节点，则最后会遍历到公共节点上，否则遍历到null。

class Solution {
    public ListNode getIntersectionNode ( ListNode headA, ListNode headB ) {
        if ( headA == null || headB == null ) {
            return null;
        }
        ListNode a = headA;
        ListNode b = headB;
        while ( a != b ) {
            // a等于null表示已经走到当前链表尾部，需要跳转到B链表开头
            // a不等于null表示还在当前链表中遍历
            // b同理
            a = a == null ? headB : a.next;
            b = b == null ? headA : b.next;
        }
        // a != null：A和B具有相同节点的话，返回节点
        // a == null，A和B无相同节点，返回null
        return a;
    }
}
