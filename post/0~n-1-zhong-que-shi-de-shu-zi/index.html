0~n-1中缺失的数字

2021-08-27

一个长度为n-1的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围0～n-1之内。在范围0～n-1内的n个数字中有且只有一个数字不在该数组中，请找出这个数字。

  • 方法1：二分法

排序数组中的搜索问题，可以尝试运用二分法来解决。最后一次交换后left=右子数组的首位元素，right=左子数组的末位元素，所以返回left。

class Solution {
    public int missingNumber ( int[] nums ) {
        int left = 0, right = nums.length - 1;
        while ( left <= right ) {
            int m = ( left + ( right - left ) ) / 2;
            if ( nums[m] == m ) {
                left = m + 1;
            } else {
                right = m - 1;
            }
        }
        return left;
    }
}

  • 方法2：抑或操作

原理在于抑或（相同为0，不同为1）操作：abb=a

class Solution {
    public int missingNumber ( int[] nums ) {
        // 因为缺失一个数字，所以数组的末尾会存在x=nums.length，令其与res抑或则会抵消
        int res = nums.length;
        for ( int i = 0; i < nums.length; i++ ) {
            res ^= nums[i];
            res ^= i;
        }
        return res;
    }
}
