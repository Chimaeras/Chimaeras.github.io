从上到下打印二叉树

2021-09-12

从上到下打印出二叉树的每个节点，同一层的节点按照从左到右的顺序打印。

  • 方法1：队列维护节点顺序（层序）

按照层序将节点入队，入队的同时将节点的值放入数组中，完全入队之后，输出数组。

class Solution {
    public int[] levelOrder ( TreeNode root ) {
        if ( root == null ) {
            return new int[0];
        }
        Queue< TreeNode > queue = new LinkedList<>();
        queue.add( root );
        ArrayList< Integer > arrayList = new ArrayList<>();

        while ( !queue.isEmpty() ) {
            TreeNode node = queue.poll();
            arrayList.add( node.val );
            if ( node.left != null ) {
                queue.add( node.left );
            }
            if ( node.right != null ) {
                queue.add( node.right );
            }
        }

        int[] res = new int[arrayList.size()];
        for ( int i = 0; i < arrayList.size(); i++ ) {
            res[i] = arrayList.get( i );
        }

        return res;
    }
}

变种

上到下按层打印二叉树，同一层的节点按从左到右的顺序打印，每一层打印到一行。

  • 方法1：同上

大致做法和上题类似，不过这题中，我们要在每一层中，将当前队列中的节点添加进数组并清空，在插入下一层的节点。同时有一个要注意的点，当我们for循环时，要将初始值取为queue.size（），然后再i--。因为如果判断条件是i<queue.size()的话，每次节点出队之后，队列的长度会变。

class Solution {
    public List< List< Integer > > levelOrder ( TreeNode root ) {
        //创建结点队列
        Queue< TreeNode > queue = new LinkedList<>();
        //创建结果数组
        List< List< Integer > > res = new ArrayList<>();
        //如果根节点不为空，加入队列
        if ( root != null ) {
            queue.add( root );
        }
        //队列不为空说明未循环完
        while ( !queue.isEmpty() ) {
            //创建当前层的结点的数组
            List< Integer > temp = new ArrayList<>();
            //取当前队列的长度，依次遍历结点
            for ( int i = queue.size(); i > 0; i-- ) {
                //结点出队
                TreeNode t = queue.poll();
                temp.add( t.val );
                if ( t.left != null ) {
                    queue.add( t.left );
                }
                if ( t.right != null ) {
                    queue.add( t.right );
                }
            }
            res.add( temp );
        }
        return res;
    }
}

变种

请实现一个函数按照之字形顺序打印二叉树，即第一行按照从左到右的顺序打印，第二层按照从右到左的顺序打印，第三行再按照从左到右的顺序打印，其他行以此类推。

  • 方法1：同上+双端队列

按照之子型进行打印的话，即第一层是从左到右，第二层是从右到左，我们可以利用双向队列来存储元素，当奇数层的时候我们从队列的尾部进行添加；偶数层的时候从队列的头部进行添加

class Solution {
    public List< List< Integer > > levelOrder ( TreeNode root ) {
        //创建结点队列
        Queue< TreeNode > queue = new LinkedList<>();
        //创建结果数组
        List< List< Integer > > res = new ArrayList<>();
        //如果根节点不为空，加入队列
        if ( root != null ) {
            queue.add( root );
        }
        //队列不为空说明未循环完
        while ( !queue.isEmpty() ) {
            //创建当前层的结点的数组
            LinkedList< Integer > temp = new LinkedList<>();
            //取当前队列的长度，依次遍历结点
            for ( int i = queue.size(); i > 0; i-- ) {
                //结点出队
                TreeNode t = queue.poll();
                // 偶数层 -> 队列头部
                // 奇数层 -> 队列尾部
                if ( res.size() % 2 == 0 ) {
                    temp.addLast( t.val );
                } else {
                    temp.addFirst( t.val );
                }
                if ( t.left != null ) {
                    queue.add( t.left );
                }
                if ( t.right != null ) {
                    queue.add( t.right );
                }
            }
            res.add( temp );
        }
        return res;
    }
}

  • 方法2：Collections集合的翻转函数

因为List继承了Collections集合，其实现了翻转函数，当奇数层的时候，我们将当前层数组进行翻转即可

class Solution {
    public List< List< Integer > > levelOrder ( TreeNode root ) {
        Queue< TreeNode > queue = new LinkedList<>();
        List< List< Integer > > res = new ArrayList<>();
        if ( root != null ) {
            queue.add( root );
        }
        while ( !queue.isEmpty() ) {
            List< Integer > tmp = new ArrayList<>();
            for ( int i = queue.size(); i > 0; i-- ) {
                TreeNode node = queue.poll();
                tmp.add( node.val );
                if ( node.left != null ) {
                    queue.add( node.left );
                }
                if ( node.right != null ) {
                    queue.add( node.right );
                }
            }
            if ( res.size() % 2 == 1 ) {
                Collections.reverse( tmp );
            }
            res.add( tmp );
        }
        return res;
    }
}
