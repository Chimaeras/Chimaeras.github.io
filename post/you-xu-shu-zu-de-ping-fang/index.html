有序数组的平方

2021-09-03

给定一个按非递减顺序排序的整数数组A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序

  • 方法1：直接排序

class Solution {
    public int[] sortedSquares ( int[] nums ) {
        for ( int i = 0; i < nums.length; i++ ) {
            nums[i] = nums[i] * nums[i];
        }
        Arrays.sort( nums );
        return nums;
    }
}

  • 方法2：双指针

由于源数组已经排好序了，假设mid为正负的分界线，mid之前的数会随着平方越来越小，mid之后的数会随着平方而变大。采用双指针的操作，左指针和右指针同时往中间走，判断何者更大，将更大的数字添加进答案数组。

class Solution {
    public int[] sortedSquares ( int[] nums ) {
        int size = nums.length;
        int[] res = new int[size];
        int i = 0, j = size - 1, index = size - 1;
        while ( i < j ) {
            if ( nums[i] * nums[i] > nums[j] * nums[j] ) {
                res[index] = nums[i] * nums[i];
                i++;
            } else {
                res[index] = nums[j] * nums[j];
                j--;
            }
            index--;
        }
        return res;
    }
}
