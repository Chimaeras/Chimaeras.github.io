顺时针打印矩阵

2021-09-07

输入一个矩阵，按照从外向里以顺时针的顺序依次打印出每一个数字。

  • 方法1:模拟前进法

  • 顺着左上角，模拟顺时针前进，当遇到边界的时候，进入下一个方向。

  • 使用一个辅助数组来判断当前位置是否访问过

  • 总路径长度为数组元素的总和

class Solution {
    public int[] spiralOrder ( int[][] matrix ) {
        if ( matrix == null || matrix.length == 0 || matrix[0].length == 0 ) {
            return new int[0];
        }

        int m = matrix.length;
        int n = matrix[0].length;

        // 记录是否被访问过
        boolean[][] visit = new boolean[m][n];

        // 记录总路径长度
        int total = m * n;

        // 结果数组
        int[] res = new int[total];

        // 实时记录索引
        int i = 0, j = 0;

        // 定义方向 右 下 左 上
        int[][] dir = { { 0, 1 }, { 1, 0 }, { 0, -1 }, { -1, 0 } };
        // 决定前进方向
        int dirIndex = 0;

        for ( int k = 0; k < total; k++ ) {
            res[k] = matrix[i][j];
            // 修改标记，使其表示被访问过
            visit[i][j] = true;

            // 下一步要走的方向
            int nextI = i + dir[dirIndex][0];
            int nextJ = j + dir[dirIndex][1];

            // 边界条件，如果超出边界了，就改变前进方向
            if ( nextI < 0 || nextI >= m || nextJ < 0 || nextJ >= n || visit[nextI][nextJ] ) {
                dirIndex = ( dirIndex + 1 ) % 4;
            }

            // 确保不会越界之后，再修改下一步的坐标
            i += dir[dirIndex][0];
            j += dir[dirIndex][1];
        }
        return res;
    }
}

  • 方法2：分层模拟

将整个矩阵分成一层一层来进行操作，首先打印最外层的元素，然后依次打印内层的元素。

class Solution {
    public int[] spiralOrder ( int[][] matrix ) {
        if ( matrix == null || matrix.length == 0 || matrix[0].length == 0 ) {
            return new int[0];
        }

        int m = matrix.length, n = matrix[0].length;
        int[] res = new int[m * n];
        int index = 0;
        int left = 0, right = n - 1, top = 0, bottom = m - 1;

        while ( left <= right && top <= bottom ) {
            // 向左移动
            for ( int column = left; column <= right; column++ ) {
                res[index++] = matrix[top][column];
            }
            // 向下移动
            for ( int row = top + 1; row <= bottom; row++ ) {
                res[index++] = matrix[row][right];
            }

            // 因为内层中可能存在某些层只有一行，所以要判断是否需要向左和向上移动
            if ( left < right && top < bottom ) {
                //向左移动
                for ( int column = right - 1; column > left; column-- ) {
                    res[index++] = matrix[bottom][column];
                }
                // 向上移动
                for ( int row = bottom; row > top; row-- ) {
                    res[index++] = matrix[row][left];
                }
            }
            // 修改定点，开始内层循环
            left++;
            right--;
            top++;
            bottom--;
        }
        return res;
    }
}
