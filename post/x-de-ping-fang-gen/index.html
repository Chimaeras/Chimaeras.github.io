x的平方根

2021-09-18

题目：实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

  • 解法1：

class Solution {
    public int mySqrt ( int x ) {
        int sqrt = 0;
        int i = 1;
        if ( x == 0 ) {
            return 0;
        }
        while ( ( x / i ) >= i ) {
            //当i*i<x时，记录sqrt，不断更新直到i*i>x
            sqrt = i;
            i++;
        }
        return sqrt;
    }
}

  • 解法2：

class Solution {
    public int mySqrt(int x) {
        int l = 0, r = x, ans = -1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if ((long) mid * mid <= x) {
                ans = mid;
                l = mid + 1;
            } else {
                r = mid - 1;
            }
        }
        return ans;
    }
}
