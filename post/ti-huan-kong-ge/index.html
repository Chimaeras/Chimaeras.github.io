替换空格

2021-09-12

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

题目描述

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。

  • 方法1：库函数

class Solution {
    public String replaceSpace ( String s ) {
        return s.replace( " ", "%20" );
    }
}

  • 遍历拼接

字符串是不可变对象，要使用StringBuilder拼接字符串，再转换回去。

class Solution {
    public String replaceSpace ( String s ) {
        StringBuilder res = new StringBuilder();
        for ( Character c : s.toCharArray() ) {
            if ( c == ' ' ) {
                res.append( "%20" );
            } else {
                res.append( c );
            }
        }
        return res.toString();
    }
}
