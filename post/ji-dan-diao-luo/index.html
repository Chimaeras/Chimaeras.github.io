鸡蛋掉落

2021-09-22

你将获得 K 个鸡蛋，并可以使用一栋从 1 到 N  共有 N 层楼的建筑，每个蛋的功能都是一样的，如果一个蛋碎了，你就不能再把它掉下去。

你知道存在楼层 F ，满足 0 <= F <= N 任何从高于 F 的楼层落下的鸡蛋都会碎，从 F 楼层或比它低的楼层落下的鸡蛋都不会破。

每次移动，你可以取一个鸡蛋（如果你有完整的鸡蛋）并把它从任一楼层 X 扔下（满足 1 <= X <= N）。

你的目标是确切地知道 F 的值是多少。无论 F 的初始值如何，你确定 F 的值的最小移动次数是多少？

解法1：动态规划+二分查找（java）

class Solution {
    public int superEggDrop ( int k, int n ) {
        int[][] dp = new int[n + 1][k + 1];
        //i表示楼层，j表示鸡蛋
        for ( int i = 1; i <= n; i++ ) {
            //当只有1个鸡蛋却要检查N层楼时，只能线性扫描N层楼
            dp[i][1] = i;
        }
        for ( int j = 1; j <= k; j++ ) {
            //在第一层，无论有多少个鸡蛋 都只用一次检查
            dp[1][j] = 1;
        }

        //后续节点通过binary函数算出来
        for ( int i = 2; i <= n; i++ ) {
            for ( int j = 2; j <= k; j++ ) {
                dp[i][j] = binary( i, j, dp );
            }
        }
        return dp[n][k];
    }

    public int binary ( int floors, int eggs, int[][] dp ) {
        int left = 1, right = floors;
        while ( left < right ) {
            //定义中间节点
            int mid = ( left + right ) / 2;
            //碎了：楼层-1，鸡蛋-1
            int broken = dp[mid - 1][eggs - 1];
            //没碎：楼层+1，鸡蛋不变
            int not_broken = dp[floors - mid][eggs];
            //如果没碎>碎了
            if ( not_broken > broken ) {
                //左边界变为中间值+1
                left = mid + 1;
            } else {
                //更新右边值
                right = mid;
            }
        }
        return Math.max( dp[right - 1][eggs - 1], dp[floors - right][eggs] ) + 1;
    }
}

方法2：数学法

class Solution {
    public int superEggDrop ( int k, int n ) {
        if ( n == 1 ) {
            return 1;
        }
        int[][] f = new int[n + 1][k + 1];
        for ( int i = 1; i <= k; ++i ) {
            f[1][i] = 1;
        }
        int ans = -1;
        for ( int i = 2; i <= n; ++i ) {
            for ( int j = 1; j <= k; ++j ) {
                f[i][j] = 1 + f[i - 1][j - 1] + f[i - 1][j];
            }
            if ( f[i][k] >= n ) {
                ans = i;
                break;
            }
        }
        return ans;
    }
}
