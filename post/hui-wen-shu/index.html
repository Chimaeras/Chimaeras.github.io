回文数

2021-09-04

题目：判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数

  • 解法1：

class Solution {
    public boolean isPalindrome ( int x ) {
        if ( x < 0 ) {
            return false;
        }
        else {
            int[] b = new int[10];
            int j = 0, i = 0, t;
            while ( x != 0 ) {
                b[i] = x % 10;
                x = x / 10;
                i++;
            }
            t = i - 1;
            while ( j < t ) {
                if ( b[j++] != b[t--] ) {
                    //j为数组开头，t为数组结尾
                    return false;
                }
            }
            return true;
        }
    }
}
  • 分析：将x的每一位存放到数组b中，再从b的开头和b的结尾向中间逼近。若顺序和倒序相等，则为回文数。

  • 解法2：

class Solution {
    public boolean isPalindrome ( int x ) {
        if ( x < 0 || ( x != 0 && x % 10 == 0 ) ) {
            return false;
        }
        int rx = 0;
        while ( x > rx ) {
            rx = rx * 10 + x % 10;
            x /= 10;
        }
        return x == rx || x == rx / 10;
    }
}
  • 分析：通过循环生成x的逆序数字，比较x和rx是否相等。(存在一种情况就是循环中x多除了一次10)
