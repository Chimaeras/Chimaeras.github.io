不用加减乘除做加法

2021-09-07

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

  • 方法1：位运算

两个数异或就是不进位的加法，两个数按位与是取进位

class Solution {
    public int add ( int a, int b ) {
        int sum1 = a ^ b;
        int sum2 = ( a & b ) << 1;
        return sum1 + sum2;
    }
}
