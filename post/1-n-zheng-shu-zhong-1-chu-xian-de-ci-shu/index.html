1-n整数中1出现的次数

2021-09-15

输入一个整数 n ，求1～n这n个整数的十进制表示中1出现的次数。

例如，输入12，1～12这些整数中包含1 的数字有1、10、11和12，1一共出现了5次。

  • 方法1：从低位起，循环计算当前位中1出现的次数

class Solution {
    public int countDigitOne ( int n ) {
        // 结果
        int res = 0;
        // 位因子，即pow(10,i)
        int dig = 1;
        // 高位，即当前位之前的位数
        int high = n / 10;
        // 当前位
        int cur = n % 10;
        // 低位，即当前位之后的位数
        int low = 0;

        // 当 high 和 cur 同时为 0 时，说明已经越过最高位，因此跳出
        while ( high != 0 || cur != 0 ) {
            // 根据当前位来选择对应的计算方法
            if ( cur == 0 ) {
                res += high * dig;
            } else if ( cur == 1 ) {
                res += high * dig + low + 1;
            } else {
                res += ( high + 1 ) * dig;
            }
            // 更新位信息
            // 低位 = low + pow(10,i) * cur
            // 将 cur 加入 low ，组成下轮 low
            low += cur * dig;
            // 下轮 cur 是本轮 high 的最低位
            cur = high % 10;
            // 将本轮 high 最低位删除，得到下轮 high
            high /= 10;
            // 位因子每轮 × 10
            dig *= 10;
        }
        return res;
    }
}
