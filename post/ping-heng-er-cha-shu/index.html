平衡二叉树

2021-09-13

输入一棵二叉树的根节点，判断该树是不是平衡二叉树。如果某二叉树中任意节点的左右子树的深度相差不超过1，那么它就是一棵平衡二叉树。

  • 方法1：从底至上后续遍历+剪枝

class Solution {
    public boolean isBalanced ( TreeNode root ) {
        return len( root ) != -1;
    }

    public int len ( TreeNode root ) {
        // 节点为空直接返回
        if ( root == null ) {
            return 0;
        }
        // 若左/右子树中存在不平衡二叉树，直接返回，减少操作
        int left = len( root.left );
        if ( left == -1 ) {
            return -1;
        }
        int right = len( root.right );
        if ( right == -1 ) {
            return -1;
        }
        // 如果左右子树的深度差小于2，即为平衡二叉树；否则返回-1
        return Math.abs( left - right ) < 2 ? Math.max( left, right ) + 1 : -1;
    }
}

  • 方法2：递归判断

思路和 “ 二叉树的深度 ”类似，先同一个函数获取以当前节点为根节点的树的深度，然后若root为平衡二叉树，需要满足几个条件：

  • 当前节点满足左右子树深度差小于2
  • 左节点为平衡二叉树
  • 右节点为平衡二叉树
class Solution {
    public boolean isBalanced ( TreeNode root ) {
        if ( root == null ) {
            return true;
        }
        return ( Math.abs( len( root.left ) - len( root.right ) ) <= 1 && isBalanced( root.left ) && isBalanced( root.right ) );
    }

    public int len ( TreeNode root ) {
        if ( root == null ) {
            return 0;
        }
        return 1 + Math.max( len( root.left ), len( root.right ) );
    }
}
