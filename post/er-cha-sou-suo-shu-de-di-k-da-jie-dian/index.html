二叉搜索树的第k大节点

2021-09-14

给定一棵二叉搜索树，请找出其中第k大的节点。

  • 方法1：数组存储节点，再遍历

class Solution {
    int size = 0;
    ArrayList< Integer > arrayList = new ArrayList<>();

    public int kthLargest ( TreeNode root, int k ) {
        dfs( root );
        Arrays.sort( arrayList.toArray() );
        return arrayList.get( arrayList.size() - k );
    }

    public void dfs ( TreeNode root ) {
        if ( root == null ) {
            return;
        }
        if ( root.left != null ) {
            dfs( root.left );
        }
        arrayList.add( root.val );
        if ( root.right != null ) {
            dfs( root.right );
        }
    }
}

  • 方法2：逆序递归遍历

二叉搜索树的中序遍历是递增序列（左、中、右），所以中序遍历倒序为递减序列（右、中、左）。利用全局变量k记录当前已经访问节点的个数，res记录结果值。每遍历一个点就减一。

每一趟遍历中有三个工作：

  • 提前返回： 若 k = 0 ，代表已找到目标节点，无需继续遍历，因此直接返回；

  • 统计序号： 执行 k = k - 1 （即从 k 减至 0 ）；

  • 记录结果： 若 k = 0 ，代表当前节点为第 k 大的节点，因此记录 res = root.val；

class Solution {
    int res, k;

    public int kthLargest ( TreeNode root, int k ) {
        this.k = k;
        dfs( root );
        return res;
    }

    public void dfs ( TreeNode root ) {
        if ( root == null ) {
            return;
        }
        dfs( root.right );
        if ( k == 0 ) {
            return;
        }
        k--;
        if ( k == 0 ) {
            res = root.val;
        }
        dfs( root.left );
    }
}
