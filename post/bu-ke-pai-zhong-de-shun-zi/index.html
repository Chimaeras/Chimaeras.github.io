扑克牌中的顺子

2021-09-11

从若干副扑克牌中随机抽 5 张牌，判断是不是一个顺子，即这5张牌是不是连续的。2～10为数字本身，A为1，J为11，Q为12，K为13，而大、小王为 0 ，可以看成任意数字。A 不能视为 14。

  • 方法1：set存储不重复的元素

class Solution {
    public boolean isStraight ( int[] nums ) {
        HashSet< Integer > set = new HashSet<>();
        int max = 0, min = 14;
        for ( int num : nums ) {
            // 将大于0的数添加进去
            if ( num > 0 ) {
                // 加进来之前判断是否已经存在，存在说明重复，不符合
                if ( set.contains( num ) ) {
                    return false;
                }
                set.add( num );
                // 更新最大最小值
                max = Math.max( max, num );
                min = Math.min( min, num );
            }
        }
        return max - min < 5;
    }
}

  • 方法2：排序+遍历

class Solution {
    public boolean isStraight ( int[] nums ) {
        Arrays.sort( nums );
        int joker = 0;
        for ( int i = 0; i < 4; i++ ) {
            // 统计鬼的数量
            if ( nums[i] == 0 ) {
                joker++;
            } else if ( nums[i] == nums[i + 1] ) {
                // 存在重复
                return false;
            }
        }
        // 最大-最小<5
        return nums[nums.length - 1] - nums[joker] < 5;
    }
}
