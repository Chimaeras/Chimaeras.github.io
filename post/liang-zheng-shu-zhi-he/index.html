两整数之和

2021-09-18

题目：不使用运算符 + 和 - ​​​​​​​，计算两整数 ​​​​​​​a 、b ​​​​​​​之和。

  • 解法：

class Solution {
    public int getSum ( int a, int b ) {
        // 获取相加和
        int sum = a ^ b;
        // 获取进位
        int carry = ( a & b ) << 1;
        // 如果还存在进位，接着计算
        if ( carry != 0 ) {
            return getSum( sum, carry );
        }
        return sum;
    }
}
