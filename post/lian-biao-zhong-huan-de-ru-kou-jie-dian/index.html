链表中环的入口节点

2021-09-12

给定一个链表，返回链表开始入环的第一个节点。 从链表的头节点开始沿着 next 指针进入环的第一个节点为环的入口节点。如果链表无环，则返回 null。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。注意，pos 仅仅是用于标识环的情况，并不会作为参数传递到函数中。

**说明：**不允许修改给定的链表。

  • 方法1：双指针（快慢指针）

class Solution {
    public ListNode detectCycle ( ListNode head ) {
        // 头节点为空，直接返回null
        if ( head == null ) {
            return null;
        }
        ListNode slow = head, fast = head;
        while ( true ) {
            // 如果快指针指向空节点的话，肯定无环，有环的话会无限循环走下去
            if ( fast == null || fast.next == null ) {
                return null;
            }
            slow = slow.next;
            fast = fast.next.next;
            // 找到第一次相遇的节点
            if ( slow == fast ) {
                break;
            }
        }
        // 快指针指向开头
        fast = head;
        // 快慢指针同时前进
        while ( slow != fast ) {
            slow = slow.next;
            fast = fast.next;
        }
        return fast;
    }
}
