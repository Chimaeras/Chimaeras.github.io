宝石与石头

2021-09-17

题目：给定字符串J 代表石头中宝石的类型，字符串 S代表你拥有的石头。 S 中每个字符代表了一种你拥有的石头的类型，你想知道你拥有的石头中有多少是宝石。J 中的字母不重复，J 和 S中的所有字符都是字母。字母区分大小写，因此"a"和"A"是不同类型的石头。

  • 解法1：暴力遍历

class Solution {
    public int numJewelsInStones ( String jewels, String stones ) {
        char[] jChar = jewels.toCharArray();
        char[] sChar = stones.toCharArray();

        int res = 0;
        for ( char j : jChar ) {
            for ( char s : sChar ) {
                //对比两个数组的字母 若有相同项就加1
                if ( j == s ) {
                    res++;
                }
            }
        }
        return res;
    }
}
  • 分析：对比两个数组的字母 若有相同项就加1

  • 方法2：set存储宝石种类

class Solution {
    public int numJewelsInStones ( String jewels, String stones ) {
        int count = 0;
        Set< Character > set = new HashSet< Character >();
        // 放进set中，存储宝石种类
        for ( int i = 0; i < jewels.length(); i++ ) {
            char j = jewels.charAt( i );
            set.add( j );
        }
        // 查询个数
        for ( int i = 0; i < stones.length(); i++ ) {
            char s = stones.charAt( i );
            if ( set.contains( s ) ) {
                count++;
            }
        }
        return count;
    }
}

  • 方法3：数组模拟set

class Solution {
    public int numJewelsInStones ( String J, String S ) {
        int[] type = new int[256];
        // 存储宝石种类
        for ( int i = 0; i < J.length(); i++ ) {
            type[J.charAt( i )] = 1;
        }
        int ans = 0;
        // 累计宝石个数
        for ( int i = 0; i < S.length(); i++ ) {
            ans += type[S.charAt( i )];
        }
        return ans;
    }
}
