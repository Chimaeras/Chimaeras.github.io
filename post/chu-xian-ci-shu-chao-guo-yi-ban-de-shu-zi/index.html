出现次数超过一半的数字

2021-09-08

数组中有一个数字出现的次数超过数组长度的一半，请找出这个数字。

  • 方法1：排序法

因为目标数字出现次数超过一半，所以数组的中间数肯定为目标数字

class Solution {
    public int majorityElement ( int[] nums ) {
        Arrays.sort( nums );
        return nums[nums.length / 2];
    }
}

  • 方法2：摩尔投票法

因为存在某个数字出现次数超过一半，所以每个数字进行投票，最后票数多的肯定是目标数字

class Solution {
    public int majorityElement ( int[] nums ) {
        int count = 0, res = 0;
        for ( int num : nums ) {
            if ( count == 0 ) {
                res = num;
            }
            count += num == res ? 1 : -1;
        }
        return res;
    }
}
