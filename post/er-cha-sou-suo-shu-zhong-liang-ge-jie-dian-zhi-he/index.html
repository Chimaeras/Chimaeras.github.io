二叉搜索树中两个节点之和

2021-09-14

给定一个二叉搜索树的 根节点 root 和一个整数 k , 请判断该二叉搜索树中是否存在两个节点它们的值之和等于 k 。假设二叉搜索树中节点的值均唯一。

  • 方法1：队存储所有节点，遍历查找。

注意，这里不能用用res.cotains（）来进行包含查询，可能会出现 [ 1 , null , 2]。即单个点等于target的情况出现。

class Solution {
    ArrayList< Integer > res = new ArrayList<>();

    public boolean findTarget ( TreeNode root, int k ) {
        if ( root == null ) {
            return false;
        }
        dfs( root, k );
        int i = 0, j = res.size() - 1;
        while ( i < j ) {
            if ( res.get( i ) + res.get( j ) == k ) {
                return true;
            } else if ( res.get( i ) + res.get( j ) < k ) {
                i++;
            } else {
                j--;
            }
        }
        return false;
    }

    public void dfs ( TreeNode root, int k ) {
        if ( root.left != null ) {
            dfs( root.left, k );
        }
        res.add( root.val );
        if ( root.right != null ) {
            dfs( root.right, k );
        }
    }
}
