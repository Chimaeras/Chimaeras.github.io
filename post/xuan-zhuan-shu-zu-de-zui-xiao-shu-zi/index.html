旋转数组的最小数字

2021-09-07

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。输入一个递增排序的数组的一个旋转，输出旋转数组的最小元素。例如，数组 [3,4,5,1,2] 为 [1,2,3,4,5] 的一个旋转，该数组的最小值为1。

  • 二分法

和常规的二分法类似，但是有几个要注意的点：

  • 每次比较的时候，和右边进行比较，因为右边界可以确保是处于右旋转数组，但是就算给出了mid大于left，也不能保证left是处于左旋转数组中。
  • 如果 mid == right 的话，则表示在 [ left , mid ] 或 [ mid , right ]之间均为相等的值，也可能同时都满足，无法通过二分安全的缩小区间。
class Solution {
    public int minArray ( int[] numbers ) {
        int left = 0;
        int rigth = numbers.length - 1;
        while ( left < rigth ) {
            int mid = left + ( rigth - left ) / 2;
            if ( numbers[mid] < numbers[rigth] ) {
                rigth = mid;
            } else if ( numbers[mid] > numbers[rigth] ) {
                left = mid + 1;
            } else {
                rigth -= 1;
            }
        }
        return numbers[left];
    }
}
