包含min函数的栈

2021-09-11

定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

  • 双栈模拟

利用一个辅助栈来存储递减序列，栈A正常完成基本操作，栈B不断存储递减序列，更新的同时也要注意与栈A保持一致

  • 如果当前值小于等于A的栈顶，则B栈也要将此元素入栈，否则后续可能会出现栈空的情况
  • 如果A的栈顶元素和B的栈顶元素相等，那么当A出栈的时候，B也要出栈，防止本该已经出栈的元素还存留在栈B
class MinStack {
    Stack< Integer > A, B;

    public MinStack ( ) {
        A = new Stack<>();
        B = new Stack<>();
    }

    public void push ( int x ) {
        A.add( x );
        if ( B.empty() || B.peek() >= x ) {
            B.add( x );
        }
    }

    public void pop ( ) {
        if ( A.pop().equals( B.peek() ) ) {
            B.pop();
        }
    }

    public int top ( ) {
        return A.peek();
    }

    public int min ( ) {
        return B.peek();
    }
}
