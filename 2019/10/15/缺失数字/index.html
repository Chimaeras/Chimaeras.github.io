缺失数字

发表于 2019-10-15 | 分类于 LeetCode |

题目描述

给定一个包含 0, 1, 2, …, n 中 n 个数的序列，找出 0 .. n 中没有出现在序列中的那个数

  • 解法1：

  • 分析：先排序数组，然后遍历，若索引号不等于数值，则标志为1，返回

  • 解法2：

  • 分析：抑或(不同为1，相同为0)，一个元素抑或两次则不变，最后剩下出现一次的元素。

  • 解法3：

  • 分析：n记录序列里数的总和，s记录完整序列里数的总和，s-n就是缺失的数字

  • 解法4：

  • 分析：设置一个全为0的数组，将nums[]作为下标，若存在就将该项改为1.最后只剩缺失的数字作为下标的那项为0.
